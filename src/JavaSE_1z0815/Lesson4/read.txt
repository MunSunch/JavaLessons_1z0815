    Экземпляр класса - это объект, наделенный состоянием(state) и поведением(behavior), прописанными в классе.
            Structure Java's class
    package statement;
    import statement;
    Class declaration and definitions{
        Variables(State)
        Methods(Behavior)
        Constructors
        Comments
    }
    *Название public класса должно совпадать с именем файла, причем он должен быть один.
    Может и не быть public класса, но обязательно имя файла должно совпадать с именем класса*
    При наличии нескольких методов main, выполняется тот, который находится в public классе.

    Для создания экземпляра класса следует воспользоваться конструктором(смотри BankAccount). Тип данных является
ссылочным(reference). Это значит, что переменная такого типа хранит в себе не значение, как в примитивном типе, а
адрес или указатель на область в памяти со значением. И
    Переменные класса(instance var) не обязательно инициализировать, тогда переменная принимает default value: у целочисленных
+char - 0, boolean - false, дробных - 0.0, reference - null.
