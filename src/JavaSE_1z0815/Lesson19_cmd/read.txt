    Command line arguments
    При компиляции и запуске программы вручную по желанию можно указать аргументы, которые прописаны в качестве параметров
в функции main. Они указываются через пробел после имени файла как показано на скриншоте.

    Varargs
    Когда нужно прописать в параметрах функции несколько значений вовсе не обязательно прописывать массив значений, можно
воспользоваться varargs. Такая переменная объявляется тремя точками между типом и идентификатор:
                                        int ... a
    Varargs должен быть только один и должен стоять последним. И так как за капотом Java переводит такую переменную в массив,
перегруженый метод с выходным массивом будет конфликтовать с методом, содержащим такую переменную.

    Foreach loop
    Позволяет сократить часть кода за счет упрощения синтаксиса:
        for(int a: array){...}
