    ArrayList
    Arraylist - это часть коллекций, которая наряду с обычным массивом используется также часто. Находится в пакете java.util.
Главными отличиями от Array являются:
        1) Длина массива не фиксирована, она изменяется при добавлении элементов;
        2) Массив имеет тип Object, поэтому элементами такого массива могут быть только наследники класса Object, т.е. все
    классы. Из этого также следует, что элементами не могут быть примитивы.
    После создания объекта ArrayList образуется массив с capacity=10(как у StringBuilder'a). При добавлении, к примеру, одного
элемента образуется новый массив с capacity=11 и туда добавляется прошлый массив и этот элемент. Конструктор ArrayList
допускает задавать capacity вручную. Size листа, образованного дефолтным конструктором нуль.
    Кроме того, помимо capacity имеется size - количество объектов, находящихся внутри листа.

    Methods
    Приведены в классе ArrayList_method.java
    Особое внимание следует обратить на методы, отмеченные звездочкой. В своей реализации эти методы применяют метод equals(),
а он, как известно, в некоторых классах не реализован, поэтому даже, если объекты, например StringBuilder, между собой равны,
то equals() выдаст false, что ни к чему не приведёт.
