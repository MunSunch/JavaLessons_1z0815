    Flow control. Конструкции if-else и оператор Ternary
    Flow control позволяет контролировать ход выполнения программы(смотри скриншот), обеспечивается selection и looping.
Selection выставляет условия, соблюдение которых необходимо для дальнейшего хода исполнения программы. Looping представляет
цикличность исполнения кода.
    Selection in Java представлен конструкциями: if-else и switch.
    При выполнении условия, указанного после if, исполняется код, указанный в фигурных скобках. Если скобок нет, то
исполняется только первая строка кода, указанная после if, остальные инструкции после этой строки принадлежат к общему коду
 и исполняются независимо от условия if (за пояснениями обращайся к Test.java).
    При невыполнении условия if, проверяются условия "else if()", если такие имеются.
    При невыполнении условия else if, выполняется else, если такой имеется.
    Таким образом, общая структура выглядит так:

        if(){}
        else if()
        else{}

    if-else можно вкладывать друг в друга.

    Сравнение объектов между собой
    Сравнение объектов(оператор ==) примитивного типа данных происходит по значению, ссылочного типа - по адресу.
Случаются ситуации, когда объекты с разными адресами, но с одинаковым содержимым требуется сравнить как-раз таки по содержимому.
В таких случаях оператор сравнения бесполезен. Надо использовать метод equals(), определенный в классе Object и перезаписанный
в его классе-наследнике.
    Ternary operator
    Также используется для сравнения. Реализует if-else в более свернутом виде.
