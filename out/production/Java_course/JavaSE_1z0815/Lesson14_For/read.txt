    Циклы в Java
    Циклы необходимы для многократного выполнения одного и того кода. В Java существует 4 вида циклов: for, foreach, while,
do-while.
    Loop for применяется в случае, когда известно число выполнения кода. Синтаксис такой:
                                    for (инициализация; условие; обновление){
                                        Код, который нужно многократно выполнить
                                    }
    * Если фигурных скобок не будет, то зациклится только самая первая строка, следующая за for.
    В начале инициализируется начальное значение, затем оно попадает в условие, после, если условие выдает true, выполняется
код, записанный в теле, и уже в самом конце выполняется обновление стартового значения. Уже в следующей итерации вновь
проверяется условие, выполняется код, обновляется значение. Так происходит пока условие выдает true.
    Рассмотрим каждый из элементов поподробнее.
    Инициализация(Loop_for_property.java):
        1) Может и не быть. Достаточно инициализировать переменную вне цикла;
        1.2) Объявить переменную можно вне цикла, а инициализировать в цикле;
        2) Scope переменной ограничивается фигурными скобками цикла;
        3) Можно указать несколько переменных одного типа;
    Условие:
        4) Может и не быть, тогда условие=true(бесконечный цикл);
        5) Можно и использовать булевы операторы;
        6) Использовать несколько переменных;
    Обновление:
        7) Может и не быть(бесконечный цикл);
        8) Можно записать в выполняемый код, и не указывать его в инициализации. Суть не поменяется;
        9) Можно вставить какие-либо функции.

    Unreachable statement - это бессмысленные конструкции, которые компилятор не пропускает. Смотри скриншот и Unreachable.java

    Loop statement
    break - выход из цикла
    continue - пропуск текущей итерации, переход на следующую.

    Nested loop
    Циклы можно вкладывать друг в друга. Причем в Java возможно прервать или пропустить итерацию внешнего цикла из внутреннего
цикла(Nested_loop.java).
